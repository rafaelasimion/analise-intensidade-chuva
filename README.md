# 🌧️ Dashboard - Análise de Intensidade de Chuva (ESALQ)

Dashboard interativo desenvolvido em **Python + Streamlit** para análise de dados pluviométricos da estação automática da ESALQ/USP.

O sistema realiza:

- download automático dos dados da estação
- limpeza de erros do sensor
- cálculo de intensidade de chuva
- cálculo de acumulado diário
- análise mensal
- visualização em gráficos interativos
- exportação de dados em CSV
